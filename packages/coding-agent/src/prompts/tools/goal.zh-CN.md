管理当前激活的目标模式目标。

使用单个 `op` 字段：
- `create` 启动一个目标。需要 `objective`；可选的 `token_budget` 必须为正数。仅当不存在目标且没有目标处于暂停状态时使用。
- `get` 返回当前目标（激活或暂停）以及剩余的令牌预算。
- `resume` 重新激活一个已暂停的目标，以便工作可以继续。
- `complete` 在你已根据当前证据核验每一项交付物之后，将该目标标记为完成。
- `drop` 丢弃当前目标而不将其完成。

当预算较低或一轮即将结束时，NEVER 调用 `complete`。只有在目标确实已经完成并已核验时才调用它。
如果 `get` 显示有一个已暂停的目标，则在继续处理它之前调用 `resume`。
