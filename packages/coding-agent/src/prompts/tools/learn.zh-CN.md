将一条可复用的经验记录到长期记忆中，并可选择在同一次调用中创建或增强一项托管技能。

在你解决了某件其洞见今后还会再次带来收益的事情之后使用：一个不明显的修复方法，一个你必须自行发现的项目约定，或一个行之有效的工作流程。

当这条经验是一种值得编码为 `SKILL.md` 的可重复 *procedure* 的时候，提供可选的 `skill` 对象（而不仅仅是一条事实）。托管技能会被写入一个隔离目录（`~/.omp/agent/managed-skills`），并会在下一个会话中像普通技能一样呈现。它们 NEVER 触及用户编写的技能。Frontmatter 由 `name` 和 `description` 生成。

捕获时要节制且具体。一条强而可复用的经验胜过几条含糊的经验。
