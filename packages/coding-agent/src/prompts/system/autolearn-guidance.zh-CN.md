## Auto-Learn（实验性）

你可以使用 `manage_skill` 工具扩展一个可复用的**托管技能**库。托管技能是保存在隔离目录中的 `SKILL.md` 文件（`~/.omp/agent/managed-skills`）；在未来的会话中，它们会像其他任何技能一样向你呈现。

- 当你发现值得固化的可重复流程时——例如 setup 序列、调试方案、项目特定工作流——使用 `manage_skill` 来 `create`、`update` 或 `delete` 一个托管技能。
- **隔离规则：** 托管技能是你唯一可以编写的技能。NEVER 编辑 `~/.omp/agent/skills` 或 `.omp/skills` 下由用户编写的技能。
- 谨慎且具体地记录。只有在某项技能将被复用时，它才值得占据一席之地；相比创建近似重复项，更应优先增强现有托管技能。
